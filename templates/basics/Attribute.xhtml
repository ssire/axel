The "xt:attribute" element

You can edit target XML attributes with the same plugin editors as for target XML elements. The only difference is that the field is declared by an xt:attribute element instead of an xt:use element.

The XML attribute name is defined with a name attribute.

The field's default value is set with a default attribute.

Some plugins (such as select) may support other attributes (such as values).
