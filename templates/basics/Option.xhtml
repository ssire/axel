The "option" attribute of the "xt:use" or "xt:attribute" elements

You can make a single entry field (primitive editor) optional by using an option attribute when you insert it with an xt:use or an xt:attribute XTiger element.

Limitations : currently you can only use the option attribute on a plugin that explicitly supports it. You cannont use it to make optional an xt:use that includes a list of component types, or a component type which is not a primitive component type.
