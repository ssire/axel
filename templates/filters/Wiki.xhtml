The "wiki" filter

The "wiki" filter turns a text entry field into a rich text entry field. For that purpose it interprets special character conventions (Wiki conventions) into bold, italic, verbatim and link formatting commands.

The conventions are the following ones :

  • **bold** generates some bold text
  • __italic__ generates some italic text
  • ''verbatim'' generates some verbatim text
  • ==link to a url==(http://www.wikipedia.com) generates some link to a url

This filter is compatible with the "text" plugin.

Parameters

    • 'wiki_lang' (default|html|span) : choose between 3 XML markup syntax for XML serialization
      • the default syntax serializes to a forrest of <Fragment>, <Fragment FragmentKind="verbatim|important"> and <Link> elements
      • the span syntax serializes to a forrest of <span>, <span class="verbatim|important"> and <a> elements
      • the html syntax serializes to a forrest of mixed content with text(), <strong>, <em> and <a> elements
